# devlog-yew
[](https://github.com/wHoIsDReAmer/devlog-yew/blob/main/screenshots/main.png)
Private devlog website built with yew-rs

# Getting start with rust
```
cargo install trunk
rustup target add wasm32-unknown-unknown

trunk serve --open
```

# Build
```
trunk build --release
```
